# e-commerce-api

## API Endpoints

###  `/api/getproducts`
get all the products

### `/api/getproduct/:id `
get products by id

### `/api/createproduct`
(POST) create a new product

### `/api/updateproduct/:id` 
(UPDATE) update existing product

### `/api//deleteproduct/:id`
(DELETE) Delete product using ID
